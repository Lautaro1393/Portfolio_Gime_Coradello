# Project Brief: Alma Model Portfolio

## 1. Project Overview
A high-fidelity digital portfolio for **Alma**, designed to showcase versatility across professional modeling sessions. The project transitions between minimalist editorial aesthetics and bold, high-contrast visual styles.

## 2. Core Objectives
- **Professionalism**: Provide a clear, accessible "Composite" section for casting directors.
- **Versatility**: Showcase different photographic styles (Urban, Studio, Creative).
- **Conversion**: Facilitate immediate booking via integrated WhatsApp, Instagram, and Email CTAs.

## 3. Design Directions

### Direction A: High-End Editorial (The Baseline)
- **Aesthetic**: Minimalist, clean, airy.
- **Typography**: Playfair Display (Serif) for headlines; sans-serif for metadata.
- **Color Palette**: Light, neutral backgrounds (#f9f9f9) to keep focus on photography.
- **Layout**: Structured grid with generous whitespace.

### Direction B: Dark Mode (Cinematic)
- **Aesthetic**: High contrast, moody, premium.
- **Color Palette**: Deep Black and Charcoal Grey variants.
- **Impact**: Enhances color saturation in photography, particularly effective for studio lighting.

### Direction C: Neo-Pop Radical (Maximalist)
- **Aesthetic**: Bold, energetic, experimental.
- **Color Palette**: Electric Fuchsia, Lime Green, and Deep Blue.
- **Layout**: Asymmetric grids and color-blocked sections that treat the UI as an extension of the creative photography.

## 4. Feature Set
- **Dynamic Hero Section**: Large-scale branding and primary "Booking" CTA.
- **Session Galleries**: 
    - *Urban Style*: Street-wear and outdoor photography.
    - *Studio / Fashion*: Controlled lighting and high-fashion looks.
    - *Creative / Color*: Artistic, high-energy sessions.
- **Interactive Composite**: Digital measurement card (Height, Bust, Waist, Hips, Eyes, Hair).
- **Global Navigation**: Portfolio, Sessions, Measurements, Contact.
- **Contact Footer**: Direct social and professional links.

## 5. Technical Specifications
- **Framework**: Tailwind CSS / HTML5.
- **Responsiveness**: Desktop-first with mobile-optimized navigation.
- **Design System**: "High-End Editorial Portfolio" tokens for consistent spacing and typography across variants.
